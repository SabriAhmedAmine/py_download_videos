# py Prerequisites
Python 3.x installed.

yt-dlp library: Install via pip install yt-dlp.

FFmpeg: Required for merging video and audio streams. Ensure it's installed and added to your system's PATH. (extracte,copy bin folder into a new folder that you create in C drive named ffmpeg,copy the new folder path and add it to the enviroment variables)

Internet connection: To download videos/playlists from YouTube.
